### Para execução do webscrapping em python:
pip install bs4, selenium, webdriver_manager

### Backend: 
npm install -i

nodemon server.js

### Frontend:
npm install -i

npm start

### Banco de dados
A aplicação utiliza PostgreSQL, o script necessário para a criação do banco está disponível em Código-Fonte/Scripts
